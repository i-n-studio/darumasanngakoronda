# だるまさんがころんだ - Monaca/Cordova

このリポジトリは、Monaca (Cordova) を用いたスマホ/PC対応のミニゲーム「だるまさんがころんだ」のプロトタイプです。

## 機能
- 傾きセンサー（DeviceMotion / DeviceOrientation）を使用したバランスチャレンジ
- 「進む」ボタン長押しで移動、鬼の『振り向き』で停止判断
- バランス成功でバフ、失敗でデバフ
- デバッグ表示 / Web Audio でのBGM簡易実装

## 使い方
1. Monaca か Cordova 環境へアップロード
2. `config.xml` にあるプラグイン（cordova-plugin-device-motion 等）を Monaca のプロジェクトで有効化してください
3. ブラウザで実行する場合、センサーイベントがサポートされていない端末ではキーボード（←/→）で傾きの代替が使えます

## ファイル構成
- `index.html` - メインのHTML
- `css/style.css` - UIスタイル
- `js/game.js` - ゲームロジックとセンサー読み取り
- `config.xml` - Cordova用設定（Monacaからのビルドに使用）
- `docs/開発状況.md` - 開発状況のドキュメント
- `docs/ゲーム仕様書.md` - 更新されたゲーム仕様書

## バージョン管理 (Git)
本プロジェクトでは `git` を利用してスプリント管理とドキュメントの更新を行います。

ワークフローの概要:
- ブランチは `feature/` `develop` `main` を基本に運用。
- スプリント終了時は `docs/スプリント記録テンプレート.md` を用いて `docs/sprints/` にスプリントログを作成し、コミットしてください。
- `scripts/update_dev_status.sh "Sprint名"` でテンプレートファイルを生成できます（Mac環境）

実機テスト結果は `docs/開発状況.md` に追記して運用します。iOSとAndroidでの差分を明確に示すことを推奨します。
